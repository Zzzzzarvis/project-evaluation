# Dune Dashboard 手动步骤（每个项目必做）

## 1. 找到自动创建的查询
1. 登录 Dune（个人账号）。
2. 进入 **Queries** 页面。
3. 使用关键词搜索：项目名称或合约地址。
4. 打开本次评估自动创建的查询（名称通常以项目代号开头）。

## 2. 执行查询并确认结果
1. 打开查询后点击 **Run**。
2. 检查输出是否有数据（行数、时间范围、数值合理性）。
3. 如需调整时间窗口，修改参数后重新运行。

## 3. 创建 Dashboard 并添加查询
1. 进入 **Dashboards** 页面 → 点击 **New Dashboard**。
2. 命名建议：`<项目名> - Listing Evaluation`。
3. 点击 **Add Visualization** → 选择刚才的查询。
4. 按以下顺序添加（推荐布局）：
   - 交易量趋势
   - 活跃地址/持币人数
   - 资金流入流出
   - 持币分布/集中度
   - LP 深度与变化

## 4. 导出与引用
1. 在 Dashboard 页面复制链接。
2. 将链接填入 `decision_notes.md` 的“证据链接”。
3. 在对外报告中引用关键图表结论并附链接。

## 5. 每次评估的检查清单
- 查询是否执行成功
- 数据时间范围是否正确
- 是否已添加所有核心图表
- Dashboard 链接已保存到内部记录
